# 撒砂系统 · 系统控制中心 Demo

**定位：系统控制中心 / 系统中台（V2 + 新补充）** — 数据交互与任务串连；中心大屏、移动终端由 **中台 Web 配置 + 推送**。

## 对齐新补充 §2.2 菜单结构

| 一级菜单 | 新补充对应 | 要点 |
|----------|------------|------|
| 数据交互 | §1 系统数据同步 | 七大模块对接 + 大屏/移动推送配置 + 接口监控 |
| 数据展示 | §2 多源采集可视化 | 含 **故障追溯、设备台账** |
| 任务编排 | §3 作业任务编排与管控 | 含 **智能调度、全流程管控、联动处置** |
| 数据分析 | §4.1 车载数据分析 | 独立 **车载数据分析** 页 |
| 智能模型 | §4.2–4.3 模型/撒砂模型 | 部署/训练/预测/评估 |
| 砂数据全生命周期 | §5 | 5.1 建档；**5.2–5.4 均为子系统接口对接数据（只读汇聚）** |
| 预警中心 | §7 异常报警 | 分级/推送/处置/追溯/联动 |
| 系统管理 | §6 | 用户/权限/参数/日志/维护/接口 |

## 三端展示（§2.1）

- **系统界面**：本 Web Demo（任务编排、配置、管理）
- **中心大屏**：`/exchange/screen` 配置推送 → `/display/screen` 预览
- **移动终端**：`/exchange/mobile` 配置推送 → `/system/mobile` 权限策略

## 快速启动

```bash
cd demo
npm install
npm run dev
```

访问 http://localhost:3003
